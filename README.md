# 每日新闻推送工作流 - 第一版

这个项目用于每天自动采集新闻、AI 筛选总结、生成 Markdown/HTML 简报，并推送到飞书。

## 你需要填写

1. 复制 `.env.example` 为 `.env`
2. 填入 `OPENAI_API_KEY`
3. 如果飞书机器人开启了签名校验，填入 `FEISHU_SECRET`

## 第一版范围

- 新闻领域：国际政治、全球经济、AI 行业上下游、网络安全、国内出海动态
- 新闻语言：中文为主；国外媒体保留英文原文信息并生成中文摘要
- 推送渠道：飞书机器人
- 本地沉淀：SQLite 数据库 + Markdown/HTML 简报
- 推送时间：中国时间每天 08:25
- 每期数量：15-25 条，重点新闻最多 5 条，每类 3-5 条

## 推荐执行顺序

1. 配好 `.env`
2. 检查 `config/sources.yaml`
3. 运行采集
4. 运行 AI 筛选与总结
5. 生成日报
6. 推送飞书
7. 每周生成趋势总结
